# Workshop om Geb og Spock

## Beskrivelse
Dette er tænkt som en vejledende løsning på opgaven stillet til ovenstående Workshop.

## Brug
Testen kan køres på enten Chrome eller Firefox:

    ./gradlew chromeTest
    ./gradlew firefoxTest

Eller evt. begge:

    ./gradlew test

På Windows erstattes `./gradlew` med `gradlew.bat`.
